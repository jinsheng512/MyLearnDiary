#### 2. idea批量替换空白行

批量替换空行
Ctrl+F 快捷方式打开Find/Replace工具窗
选择Regular expression项，允许查询匹配正则表达式
在Find文本框输入正则表达式：^\s*\n
点击ReplaceAll

| 转移字符 | 含义                                |
| -------- | ----------------------------------- |
| ^        | 否                                  |
| *        | 0个或多个                           |
| \s       | 字符串                              |
| \n       | 换行(LF) ，将当前位置移到下一行开头 |


故：^\s*\n 表示没有任何字符就换行的空白行
